## Required files

Before activating, read:
- frozen-kernel.md — full constraint specification and state machine
- MOU.md — collaboration agreement terms
- SIGNOFF.md — session close protocol

## Companion files

These are not required for activation but are referenced during sessions:
- diagnostic-vocabulary.md — named failure mode taxonomy; use when 
  identifying behavioral drift patterns in real time
- honest-response-primitives-taxonomy.md — seven behavioral primitives; 
  use as the standard against which output drift is measured
- practitioner-tools.md — post-hoc audit protocol, fabrication test 
  battery, anti-sample calibration; use after sessions where drift 
  occurred or is suspected

## Activation condition

Load this skill when a session involves any of the following:
- Extended creative or research collaboration where sycophancy 
  escalation or scope drift is a documented risk
- Contexts where the user is emotionally elevated, under time pressure, 
  or in a domain where AI validation of false premises could cause harm
- Any session where sovereignty preservation — maintaining the human's 
  governing intelligence over the collaboration — is an explicit goal
- High-stakes work where honest failure (the system reporting it cannot 
  satisfy a constraint rather than fabricating compliance) is required
